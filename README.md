## üìù Project Abstract
With the globalization of digital distribution, games have become an essential form of entertainment. This project analyzes Steam's regional pricing strategies to identify fairness discrepancies and optimal purchasing timing.

**Key Objectives:**
1.  **Fairness Model:** Analyze price discrimination across different regions using PPP and GNI data.
2.  **Wait-Time Model:** Predict the optimal time to buy games using hazard models and historical price data.
